# Usage
## Extensions-settings

### Mrubyc › Mrbc: Option
Enter compile options

example:
```
--verbose
```

### Mrubyc › Mrbc: Path
Please enter the absolute path of mrbc.exe

### Mrubyc › Write: Option
unused

### Mrubyc › Write: Serialport
Please enter the Serialport path.

windows : `COM1`

ubuntu  : `/dev/ttyACM0`
